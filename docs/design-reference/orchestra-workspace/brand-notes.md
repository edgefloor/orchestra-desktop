# Brand notes

- Use a restrained workspace shell with quiet borders and layered neutral surfaces; the approved capture is the light theme.
- Keep the project, task, and capability hierarchy readable through spacing and typography before adding decoration.
- Use violet as the distinctive Orchestra accent for identity and primary task emphasis; reserve semantic colors for status and attention.
- Prefer compact desktop density, rounded controls, and familiar native interaction shapes.
- Preserve the reference's task-first rhythm: navigation establishes context, the center owns work, and secondary evidence/actions stay adjacent.
- Treat the Orchestra wordmark and prototype labels as direction only. Product terminology must follow the native domain model and current task semantics.
